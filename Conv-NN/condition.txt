Вам необходимо обучить нейронную сеть для решения задачи классификации на датасете FashionMNIST:
    https://github.com/zalandoresearch/fashion-mnist

Необходимо сдать:
- код обучения модели
- обученную модель
- код по применению обученной модели на тестовых изображениях, и подсчитывающих результирующую точность
- описание решения и инструкции по запуску кода

Замечания:
- ваш код должен тренировать модель с нуля (нельзя использовать дообучение готовой модели)
- код тренировки должен использовать tensorlfow (без keras)
- внимательно следите за корректностью применения модели (особенно batchnorm, dropout)
- во время обучения нельзя использовать тестовые изображения
  (например, для early stopping-а необходимо использовать часть тренировочного сета)
